# Streamlit Learning Projects 🚀

This repository contains multiple **Streamlit applications** built while learning Streamlit.
It demonstrates different features of Streamlit including UI components, data handling, visualization, and deployment.

## ▶️ How to Run the App

1. Clone the repository

```bash
git clone https://github.com/anuradhasingh0423-ship-it/Streamlit.git
```

2. Navigate to the project folder

```bash
cd Streamlit
```

3. Install required packages

```bash
pip install streamlit pandas matplotlib seaborn plotly pillow
```

4. Run the Streamlit app

```bash
streamlit run app.py
```
